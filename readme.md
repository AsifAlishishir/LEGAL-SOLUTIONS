## Description

This repository contains the source code for a fictional legal firm's website. It includes a homepage with hero section, service highlights, practice areas (e.g., Business Law, Criminal Law), client testimonials, team profiles, recent cases, and a contact form. The design emphasizes aesthetics with golden accents, serif and sans-serif fonts (EB Garamond and Poppins), and responsive layouts.

The site is built from scratch using plain HTML and CSS, enhanced with Tailwind CSS for rapid styling. It's perfect as a template for law firms, attorneys, or similar professional services.

## Features

- **Responsive Design**: Fully mobile-friendly with Tailwind's utility classes.
- **Hero Section**: Engaging banner with call-to-action.
- **Practice Areas**: Grid layout showcasing legal specialties like:
  - Business Law
  - Criminal Law
  - Child Support
  - Education Law
  - Divorce Law
  - Tax Law
- **Testimonials & Team**: Carousel-style reviews and profile cards.
- **Contact & Footer**: Integrated form and subscription newsletter.
- **Icons & Fonts**: Powered by Font Awesome and Google Fonts.
- **Custom CSS**: Minimal custom styles in `solutions.css` for buttons and typography.

## Technologies Used

- **HTML5**: Structure and semantics.
- **CSS3**: Basic styling with imports for fonts.
- **Tailwind CSS**: Utility-first framework for responsive design (via CDN).
- **Font Awesome**: Icons for navigation and sections.
- **Google Fonts**: EB Garamond (serif) and Poppins (sans-serif).

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/AsifAlishishir/LEGAL-SOLUTIONS.git
   ```
2. Navigate to the project directory:
   ```
   cd legal-solutions
   ```
3. Open `index.html` in your browser – no build step required!

If you want to customize Tailwind:

- Install Tailwind via npm (optional for production):
  ```
  npm init -y
  npm install -D tailwindcss
  npx tailwindcss init
  ```
- Update `tailwind.config.js` as needed.

## Usage

- Open `index.html` directly in a browser for preview.
- Edit `index.html` for content changes.
- Modify `solutions.css` for custom styles.
- Deploy to GitHub Pages, Netlify, or Vercel for hosting.

Example: To change the primary button color, update `.btn-primary` in `solutions.css`.

```css
.btn-primary {
  border-radius: 8px;
  background-color: #b68c5a; /* Golden color – change here */
  color: white;
  padding: 18px 30px;
  font-size: 18px;
  font-weight: 600;
  cursor: pointer;
}
```
